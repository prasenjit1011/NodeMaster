| Command                              | Purpose                                        |
| ------------------------------------ | ---------------------------------------------- |
| `npx prisma init`                    | Initialize Prisma in the project               |
| `npx prisma generate`                | Generate Prisma Client                         |
| `npx prisma db push`                 | Push schema changes directly to the database   |
| `npx prisma migrate dev --name init` | Create and apply a migration in development    |
| `npx prisma migrate deploy`          | Apply pending migrations in production         |
| `npx prisma migrate reset`           | Reset database and reapply migrations          |
| `npx prisma migrate status`          | Check migration status                         |
| `npx prisma migrate diff`            | Compare database schema differences            |
| `npx prisma db pull`                 | Introspect existing database and update schema |
| `npx prisma db seed`                 | Run database seed script                       |
| `npx prisma studio`                  | Open Prisma Studio GUI                         |
| `npx prisma validate`                | Validate `schema.prisma`                       |
| `npx prisma format`                  | Format `schema.prisma`                         |
| `npx prisma version`                 | Show Prisma version                            |
| `npx prisma --help`                  | Show all available commands                    |
